🔡 Typography
We use IBM Plex Mono (weights: 400, 500, 600, 700) universally as the primary font across all apps.
IBM Plex Sans for body

🎨 Color Scheme
The applications rely on a highly-contrasted dark mode theme with a signature neon green accent.

Brand Accents

Primary / Brand (Neon Green): #01ff5e (Used for action buttons, rings, docs theme, and active text)
Backgrounds

Main Background: #0a0a0a (Deep dark gray, almost black)
Card / Surface Backgrounds: #141414 and #1a1a1a (Slightly lighter than the main background for depth)
Typography / Text Colors

Primary Text (Foreground): #ffffff and #fafafa (Clean white)
Card / Secondary Text: #d1d5db (Light gray)
Muted Text: #9ca3af (Medium gray)
Text on Primary: #0a0a0a (Dark text on the neon green backgrounds to maintain contrast)
Borders & UI Elements

Standard Borders: #374151 (Dark gray) or rgba(255, 255, 255, 0.1)
Error / Danger: #ef4444 (Red, primarily used in Dashboard validations)
